# py-bank-account-manager
Managing users and accounts.


1. Napravi signup formu za korisnika:
    - Signup trazi unos -> ime, prezime, email, broj mobitela, lozinka
        - spremi podatke u bazu

    - Login trazi unos email-a i lozinke
        - povlaci podatke iz baze

2. Napravi meni za korisnika:
    - 1. Curent balance
    - 2. Deposit money
    - 3. Withdraw money
    - 4. Exit
    * Kratiki opis funkcionalnosti:
    - 1. Curent balance ce svakom korisniku biti 0 , ali kada korisnik doda novac
        zelim da se taj 'balance' sprema u bazu ili obratno ako korisnik napravi 
        withdraw, ako nema dovoljno sredstava dobije tu poruku.
    - 2. Dodavanje novog balance-a u bazu.
    - 3. Dodavanje novog balance-a u bazu.
